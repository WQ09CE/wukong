# 全局锚点索引 (Global Anchors Index)

> 锚点是不可压缩的关键信息，无论金箍棒如何缩小，锚点始终保留。

## 使用说明

1. 新建锚点时，在对应类型下添加
2. 使用 `[Dxxx]` 格式在任何地方引用
3. 定期清理过期锚点
